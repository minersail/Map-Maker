# Map-Maker

Map Maker for original Adventure of Joe game.

Program to create maps using a paint-like editor and export them to a .txt file with the format

02, 00, 00, 00, 01

00, 07, 00, 00, 01

00, 00, 12, 00, 03

00, 05, 00, 00, 01

00, 05, 00, 00, 01
